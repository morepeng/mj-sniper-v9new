# MJ 狙擊手 v8.5 — 量化戰情室 PWA

> 量化股票分析終端 | Yahoo Finance 直連 | 無需伺服器

## 功能
- K線圖 + 技術指標（RSI / MACD / KDJ / BB / ATR / SuperTrend / OBV...）
- 六大因子評分（技術/基本面/趨勢/SMC/動能/風控）
- Dow Theory + Wyckoff 分析
- 批量掃描（最多 20 支）
- Finnhub + Alpha Vantage + Yahoo Finance 三方基本面

## 安裝
- **Android Chrome**：右上角 → 加入主畫面
- **iOS Safari**：分享 📤 → 加入主畫面

## 部署到 GitHub Pages
1. 建立 public 倉庫 `mj-dashboard`
2. 上傳所有檔案到根目錄
3. Settings → Pages → Deploy from main → /(root)
4. 網址：`https://你的帳號.github.io/mj-dashboard/`
